# Smart Network Device Analyzer

A **human-friendly network visualization tool** that turns ARP or Nmap output into a clear, interactive view of your local network. No installation required — open directly in your browser or via GitHub Pages.

---

## Live Demo

You can try it instantly without downloading anything:

[Smart Network Device Analyzer Live](https://your-username.github.io/smart-network-device-analyzer/)

*(Replace `your-username` with your GitHub username.)*

---

## Features

- Paste **ARP or Nmap output** and instantly see your network.  
- Devices grouped by **subnet**, with collapsible sections.  
- Shows each device’s:
  - IP address  
  - MAC address  
  - Vendor (from MAC prefix)  
  - Device type guess (iPhone, Android, Windows/Chromebook, Router/TV, Printer, Unknown/IoT)  
  - Network interface  
- **Color-coded, rounded cards** for easy visual identification.  
- **Filter by device type** — counts update dynamically.  
- Fully **dark mode** and modern, responsive design.  
- Works entirely in the browser — no server needed.
